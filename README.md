# nvim-config

## Necessary requirements

`brew install ripgrep`

## Key mapping

Leader key: ` ` (Space)

## Key mapping

### Search

| Shortcut | Description |
| --- | --- |
| / | Search |
| \<leader\>nh | Clear search highlights |

### Tabs

| Shortcut | Description |
| --- | --- |
| \<leader\>to | Open new tab |
| \<leader\>tx | Close tab |
| \<leader\>tn | Next tab |
| \<leader\>tp | Previous tab |
| \<leader\>tf | Open current buffer in new tab |
