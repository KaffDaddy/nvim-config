# nvim-config

## Necessary requirements

`brew install ripgrep`

## Key mapping

Leader key: ` ` (Space)

### File tree (nvim-tree)

| Shortcut    | Description                         |
|-------------|-------------------------------------|
| <leader>eo  | Open up file tree                   |
| <leader>et  | Toggles file tree                   |
| <leader>ecf | Shows file in buffer in file tree   |
